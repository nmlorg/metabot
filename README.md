# metabot
Modularized, multi-account bot.

## Quickstart
```
virtualenv -ppython3 mymetabot
cd mymetabot
. bin/activate
pip install metabot
mkdir config
python -m metabot
```

## Plugins
- [Google Calendar loader](https://pypi.org/project/metabot.calendars.google/): `pip install metabot.calendars.google`
